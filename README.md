# 💻 Library Management - Website 📚 
Library Management System Website

Welcome to the **Library Management System**! This project is designed to efficiently manage library resources, including books, users, and borrowing transactions
and **It's a simple project of my bachelor's graduation** 

## 🚀 Features
- 📖 **Book Management** - Add, edit, and remove books with ease.
- 👤 **User Management** - Register and manage library users.
- 🔄 **Borrow & Return System** - Track book loans and returns.
- 🔍 **Search & Filter** - Easily find books and users.
- 📊 **Dashboard & Analytics** - View library statistics and insights.

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP, MySQL
- **Server:** XAMPP (Apache & MySQL)


## 🛠 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Library-Management-System.git
   ```
   
2. Set up the database:
   - Import `library.sql` into MySQL.
3. Configure the database connection:
   - Edit `includes/db.php` with your database credentials.
4. Start the server:
   - Run XAMPP and start Apache & MySQL.
5. Open the project in a browser:
   ```
   http://localhost/Library-Management-System
   ```
